# Mock Client 测试帮助文档
# 用法: node tools/mock-client/src/index.js [选项]

# ========== 通用参数 ==========
# --scenario <name>          测试场景 (见下方场景列表)
# --http-base-url <url>      认证服务地址 (默认 http://127.0.0.1:3000)
# --host <ip>                游戏服务器地址 (默认 127.0.0.1)
# --port <port>              游戏服务器端口 (默认 7000)
# --chat-port <port>         聊天服务器端口 (默认 9001)
# --room-id <id>             房间ID (默认 room-default)
# --login-name <name>        登录用户名
# --password <pwd>           登录密码
# --timeout-ms <ms>          超时毫秒 (默认 5000)

# ========== 帧同步测试说明 ==========
# gameplay-roundtrip 场景输出示例：
#
# --- FRAME SYNC START ---
# [CLIENT] Sent input at frame 1: {"x":10,"y":5,"action":"tick-1"}
# [CLIENT] Sent input at frame 2: {"x":20,"y":10,"action":"tick-2"}
#
# --- RECEIVING FRAME BUNDLES ---
# [clientA]: frameId=1, fps=10, inputs=1
#   └─ [player-xxx] move: {"x":10,"y":5,"action":"tick-1"}
# [clientA]: frameId=2, fps=10, inputs=1
#   └─ [player-xxx] move: {"x":20,"y":10,"action":"tick-2"}
# [clientA]: frameId=30, fps=10, SILENT [SNAPSHOT: frame=30, state=in_game, members=2]
#
# 关键字段说明：
# - frameId: 当前帧编号
# - fps: 帧率（每秒帧数）
