# MyServer 文档索引

本目录保存当前正式设计文档和历史设计资料。当前实现状态以代码和配置为准；专题文档用于说明设计口径、边界和协作背景。

## 目录说明

- [总览](./总览/)：整体架构、主链路和跨服务边界。
- [协议与客户端](./协议与客户端/)：客户端接入、玩家协议和外部客户端协作说明。
- [游戏服与接入层](./游戏服与接入层/)：game-server、game-proxy、房间生命周期、更新灰度、玩法系统和运行时设计。
- [配置与场景](./配置与场景/)：CSV 配置表、热更状态和场景地图格式。
- [周边服务](./周边服务/)：服务注册、聊天邮件、匹配等独立能力服务。
- [后台与运维](./后台与运维/)：管理后台、生产拓扑、Room 迁移和演练手册。
- [安全与监控](./安全与监控/)：安全边界、限流、敏感操作和监控设计。
- [数据库](./数据库/)：数据库迁移与 schema 管理说明。
- [历史归档](./历史归档/)：项目初始设计稿，仅作历史资料，不作为当前实现依据。
