# 版本清单

这份文件用于集中记录当前项目采用的版本、工具链、运行环境和关键约束，避免在多个提示词文件中重复填写。

建议使用方式：

1. 先填写或核对这份文件
2. 再把其中的确定项同步回填到各个提示词中的 `<待确认>` 区块
3. 后续版本变化时，优先先更新这份文件
4. 文档之间互相引用时，统一使用相对路径
